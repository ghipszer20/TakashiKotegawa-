# TakashiKotegawa-

Backtester aiming to test the validity of a trading strategy used by highly successful day trader Takashi Kotegawa, tested on full S@P500 constituent data from 1996-2026 . Results displayed visually with graph + statistical analysis
• 25-day MA divergence entry (stock drops 20-35% below MA)
  • RSI oversold filter
  • Bollinger Band lower-band filter
  • Volume spike confirmation
  • Bullish reversal candle (close > open)
  • Tight stop-loss / take-profit / time-stop exit
  • 2% risk-per-trade position sizing
